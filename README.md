# My First Multi-page Website

A comprehensive multi-page educational platform based on ICT topics, developed as my first semester project.

## Topics Covered
- **Networking:** LAN, WAN, and MAN architectures.
- **PDLC:** Programming Development Life Cycle stages.
- **Cybersecurity:** Computer viruses and protection methods.
- **Advanced ICT:** E-commerce, Multimedia, and Network Security.

## Tech Stack
- **HTML5 & CSS3:** For structure and modern styling.
- **Three.js:** Animated WebGL grid background.
- **Google Fonts:** Orbitron and Poppins for a tech look.
- **JavaScript:** For interactive elements and navigation.

## Features
- **Modern UI:** Futuristic dark-tech theme with glassmorphism.
- **Responsive Design:** Optimized for all screen sizes.
- **Interactive Background:** High-authority animated visuals.
- **Easy Navigation:** Seamless "Back to Home" functionality.

## Credits & Special Thanks
- **Three.js:** Special thanks to the Three.js community for the amazing WebGL background components used via CDN.

## How to Run
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
