# My First Multi-page Website - ICT Semester Project

A comprehensive, multi-page educational platform developed as my first major web project. This site explores various Information and Communication Technology (ICT) topics through a modern, interactive interface.

## Project Overview
This project serves as a detailed digital guide for students to understand the building blocks of modern computing. It is designed with a high-authority "Dark-Tech" aesthetic, utilizing architectural construction logic in its UI elements.

## Core ICT Topics Explored
- **Networking Foundations:** Detailed analysis of LAN, WAN, and MAN architectures.
- **Programming Development Life Cycle (PDLC):** A step-by-step guide on software creation from planning to maintenance.
- **Cybersecurity & Threats:** Analysis of computer viruses, their types, and protection methods.
- **Digital Infrastructure:** Coverage of Multimedia, E-commerce, Network Security, and Email protocols.

## Technical Stack
- **Frontend Architecture:** HTML5 & CSS3 (Modern Flexbox/Grid layouts).
- **Typography:** Orbitron and Poppins Google Fonts for a futuristic, tech-oriented look.
- **Visual Effects:** Three.js (Grid1Background) for immersive, animated WebGL backgrounds.
- **UI Components:** Custom-styled action buttons with backdrop filters and glassmorphism effects.

## Features
- **Multi-page Navigation:** Seamless transitions between various topic-specific pages.
- **Responsive Layout:** Fully optimized for different screen sizes and devices.
- **Fast Performance:** Optimized for quick loading across all educational modules.

## How to Access
1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser to start the experience.
3. Use the "Back to Home" navigation to explore different modules.
